{\ttfamily clio} is a command line tool that interfaces with the REST api exposed by nodeop. In order to use {\ttfamily clio} you will need to have a local copy of {\ttfamily nodeop} running and configured to load the \textquotesingle{}\doxylink{classsysio_1_1chain__api__plugin}{sysio\+::chain\+\_\+api\+\_\+plugin}\textquotesingle{}.
