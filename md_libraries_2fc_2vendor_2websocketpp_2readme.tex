\chapter{Web\+Socket++ (0.7.0)}
\hypertarget{md_libraries_2fc_2vendor_2websocketpp_2readme}{}\label{md_libraries_2fc_2vendor_2websocketpp_2readme}\index{WebSocket++ (0.7.0)@{WebSocket++ (0.7.0)}}
Web\+Socket++ is a header only C++ library that implements RFC6455 The Web\+Socket Protocol. It allows integrating Web\+Socket client and server functionality into C++ programs. It uses interchangeable network transport modules including one based on raw char buffers, one based on C++ iostreams, and one based on Asio (either via Boost or standalone). End users can write additional transport policies to support other networking or event libraries as needed.

\doxysection*{Major Features }


\begin{DoxyItemize}
\item Full support for RFC6455
\item Partial support for Hixie 76 / Hybi 00, 07-\/17 draft specs (server only)
\item Message/event based interface
\item Supports secure Web\+Sockets (TLS), IPv6, and explicit proxies.
\item Flexible dependency management (C++11 Standard Library or Boost)
\item Interchangeable network transport modules (raw, iostream, Asio, or custom)
\item Portable/cross platform (Posix/\+Windows, 32/64bit, Intel/\+ARM/\+PPC)
\item Thread-\/safe
\end{DoxyItemize}
