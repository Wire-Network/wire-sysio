\subsubsection*{template$<$mp\+\_\+size\+\_\+t n, const \mbox{\hyperlink{classlibff_1_1bigint}{bigint}}$<$ n $>$ \& modulus$>$\newline
class libff\+::\+Fp\+\_\+model$<$ n, modulus $>$}
Arithmetic in the finite field F\mbox{[}p\mbox{]}, for prime p of fixed length.

This class implements Fp-\/arithmetic, for a large prime p, using a fixed number of words. It is optimized for tight memory consumption, so the modulus p is passed as a template parameter, to avoid per-\/element overheads.

The implementation is mostly a wrapper around GMP\textquotesingle{}s MPN (constant-\/size integers). But for the integer sizes of interest for libff (3 to 5 limbs of 64 bits each), we implement performance-\/critical routines, like addition and multiplication, using hand-\/optimzied assembly code.
