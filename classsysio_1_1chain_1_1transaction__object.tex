\doxysubsection{Detailed Description}
The purpose of this object is to enable the detection of duplicate transactions. When a transaction is included in a block a \doxylink{classsysio_1_1chain_1_1transaction__object}{transaction\+\_\+object} is added. At the end of block processing all transaction\+\_\+objects that have expired can be removed from the index.
