\doxysubsection{Detailed Description}
\doxylink{struct_opaque}{Opaque} data structured that holds a parsed ECDSA signature, supporting pubkey recovery.

The exact representation of data inside is implementation defined and not guaranteed to be portable between different platforms or versions. It is however guaranteed to be 65 bytes in size, and can be safely copied/moved. If you need to convert to a format suitable for storage or transmission, use the secp256k1\+\_\+ecdsa\+\_\+signature\+\_\+serialize\+\_\+\texorpdfstring{$\ast$}{*} and secp256k1\+\_\+ecdsa\+\_\+signature\+\_\+parse\+\_\+\texorpdfstring{$\ast$}{*} functions.

Furthermore, it is guaranteed that identical signatures (including their recoverability) will have identical representation, so they can be memcmp\textquotesingle{}ed.
