\doxysubsection{Detailed Description}
These are simple {\ttfamily std\+::string(const std\+::string\&)} validators that are useful. They return a string if the validation fails. \doxylink{struct_a}{A} custom struct is provided, as well, with the same user semantics, but with the ability to provide a new type name.
