\doxysubsection{Detailed Description}
\doxylink{struct_a}{A} weighted permission.

Defines a weighted permission, that is a permission which has a weight associated. \doxylink{struct_a}{A} permission is defined by an account name plus a permission name. The weight is going to be used against a threshold, if the weight is equal or greater than the threshold set then authorization will pass.
