\doxysubsection{Detailed Description}
Blockchain block header.

\doxylink{struct_a}{A} block header is defined by\+:
\begin{DoxyItemize}
\item a timestamp,
\item the producer that created it,
\item a confirmed flag default as zero,
\item a link to previous block,
\item a link to the transaction merkel root,
\item a link to action root,
\item a schedule version,
\item and a producers\textquotesingle{} schedule. 
\end{DoxyItemize}
