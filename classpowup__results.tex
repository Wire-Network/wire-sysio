\doxysubsection{Detailed Description}
The action {\ttfamily powupresult} is a no-\/op. It is added as an inline convenience action to {\ttfamily powerup} reservation. This inline convenience action does not have any effect, however, its data includes the result of the parent action and appears in its trace.
