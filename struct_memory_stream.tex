\doxysubsection{Detailed Description}
This class is mainly for being wrapped by \doxylink{class_encoded_input_stream}{Encoded\+Input\+Stream} or \doxylink{class_auto_u_t_f_input_stream}{Auto\+UTFInput\+Stream}.

It is similar to File\+Read\+Buffer but the source is an in-\/memory buffer instead of a file.

Differences between \doxylink{struct_memory_stream}{Memory\+Stream} and String\+Stream\+:
\begin{DoxyEnumerate}
\item String\+Stream has encoding but \doxylink{struct_memory_stream}{Memory\+Stream} is a byte stream.
\item \doxylink{struct_memory_stream}{Memory\+Stream} needs size of the source buffer and the buffer don\textquotesingle{}t need to be null terminated. String\+Stream assume null-\/terminated string as source.
\item \doxylink{struct_memory_stream}{Memory\+Stream} supports \doxylink{struct_memory_stream_a81dd4a86148b0af9527083cdef3d31d8}{Peek4()} for encoding detection. String\+Stream is specified with an encoding so it should not have \doxylink{struct_memory_stream_a81dd4a86148b0af9527083cdef3d31d8}{Peek4()}. \begin{DoxyNote}{Note}
implements Stream concept 
\end{DoxyNote}

\end{DoxyEnumerate}
